# AI Video Pro Studio

Ứng dụng tạo video AI hoàn chỉnh — deploy lên Vercel miễn phí.

## 🚀 Deploy lên Vercel (5 phút)

### Bước 1: Tải code về
- Download thư mục `ai-video-vercel` này về máy tính

### Bước 2: Đăng ký Vercel
- Vào [vercel.com](https://vercel.com) → Sign up bằng GitHub (miễn phí)

### Bước 3: Deploy
**Cách A — Vercel CLI (nhanh nhất):**
```bash
npm i -g vercel
cd ai-video-vercel
vercel
```

**Cách B — Upload qua web:**
1. Vào [vercel.com/new](https://vercel.com/new)
2. Kéo thả thư mục `ai-video-vercel` vào
3. Nhấn Deploy

### Bước 4: Dùng app
- Vercel cấp link dạng: `https://ai-video-pro-xxx.vercel.app`
- Mở link trên iPhone/Android/PC — hoạt động hoàn toàn!
